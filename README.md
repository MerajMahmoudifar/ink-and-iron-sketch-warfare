# Ink & Iron: Sketch Warfare ✏️⚔️

A turn-based grid strategy game with hand-drawn paper aesthetics, unique factions, command abilities, and simultaneous combat playback.

---

### 🌐 Live Demo
Play the game live at: **[https://ink-and-iron-sketch-warfare.pages.dev](https://ink-and-iron-sketch-warfare.pages.dev)**

---

## 🎮 Game Features
- **Paper & Ink Dieselpunk Art Style**: Hand-drawn canvas aesthetic with ink-sketch UI elements.
- **Turn-Based Grid Tactics**: Command units across custom and procedurally generated battle maps.
- **Faction Armories**: Choose distinct factions with specialized unit traits and abilities.
- **Field Manual & Codex**: Interactive guide covering tactics, unit types, and terrain dynamics.
- **Built-In Audio & Sound Effects**: Custom audio synthesis for battlefield action.

---

## 🛠️ Technology Stack
- **Frontend**: Vanilla HTML5, CSS3, JavaScript (ES6 Modules)
- **Rendering**: HTML5 Canvas with custom sketch rendering engine
- **Hosting**: Cloudflare Pages via Wrangler CLI

---

## 🚀 Deployment

To deploy updates to Cloudflare Pages:
```bash
npm run deploy
```
